# 委托

简单委托的构成:

1.声明委托类型

2.必须有一个方法包含了要执行的代码

3.必须创建一个委托实例

4.必须调用委托实例

## 1.声明委托类型

    委托类型实际上只是参数类型的一个列表以及一个返回类型。它规定了类型的实例能表示的操作。

## 2.必须有一个方法包含了要执行的代码

    基本思路是找到一个方法,它能做完美想做的事情,同事具有和委托类型相同的签名.确保在调用一个委托实例的时候,使用的参数完全匹配,而且能以完美希望的方式(就像普通方法调用)使用返回值.
